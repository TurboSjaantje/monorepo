Beschrijving

De zondagschool ‘De Liefde’, is een school waar kinderen les krijgen over de bijbel. Er is elke week één les na de ochtenddienst op zondag. Het is een school voor kinderen van groep 1 tot en met groep 8 met verschillende leraren voor deze groepen. De leidinggevende van deze zondagschool is op zoek naar een manier om de verschillende leraren, klassen en leerlingen te beheren in een overzichtelijk systeem. Deze leidinggevende en de leraren moeten de mogelijkheid hebben om in te loggen in dit systeem en de beschikbare acties voor hun rol uit te voeren. Dit staat duidelijker beschreven onder het kopje ‘Acties’.

Het gaat om veel verschillende leerlingen met allemaal hun eigen verhaal, sommige kinderen hebben een beperking of een jeugdtrauma waar rekening mee wordt gehouden. Voor elke leerling moet aangegeven kunnen worden of er speciale omstandigheden van toepassing zijn en als er tips zijn om hiermee overweg te gaan. Deze tips moeten te zien zijn bij het profiel van deze leerlingen en het moet mogelijk zijn om voor de verschillende speciale omstandigheden, tips toe te voegen. Deze tips worden bij alle leerlingen met dezelfde ‘beperking’ weergegeven. Ook moet er bij leerlingen aangegeven woorden als er iemand in de familie een ‘beperking’ heeft. Hier kan dan rekening mee gehouden worden.

Acties

Er is binnen deze zondagschool 1 leidinggevende die de mogelijkheid wil hebben om alles in het systeem te beheren. Hieronder vallen de volgende operaties:

  • Nieuwe leraren aanmaken.
  • De gegevens van leraren aanpassen.
  • Nieuwe klassen aanmaken.
  • Alle operaties die leraren in het systeem mogen uitvoeren.


Het moet mogelijk zijn om verschillende typen leraren aan te maken, leraren mét de mogelijkheid om leerlingen te beheren en leraren zonder de mogelijkheid om leerlingen te beheren. Onder leerlingen beheren vallen de volgende operaties:

  • Leerlingen toevoegen aan klassen.
  • Nieuwe leerlingen aanmaken.
  • De gegevens van leerlingen aanpassen.
  • Tips toevoegen voor verschillende speciale omstandigheden.

User Stories

Admin

  • 1.1: Als admin wil ik nieuwe leraren aan kunnen maken, zodat ik deze overzichtelijk weer kan geven.
  • 1.2: Als admin wil ik een leraar kunnen bewerken, zodat ik deze up-to-date kan houden.
  • 1.3: Als admin wil ik een leraar kunnen verwijderen, zodat ik odue leraren uit het systeem kan halen.
  • 1.4: Als admin wil ik een leraar kunnen bekijken, zodat ik zijn gegevens tot mijn beschikking heb.

Admin & Teacher

  • 2.1: Als Admin of Leraar wil ik nieuwe leerlingen aan kunnen maken, zodat ik deze overzichtelijk weer kan geven.
  • 2.2: Als Admin of Leraar wil ik een leerling kunnen bewerken, zodat ik deze up-to-date kan houden.
  • 2.3: Als Admin of Leraar wil ik een leerling kunnen verwijderen, zodat ik odue leerlingen uit het systeem kan halen.
  • 2.4: Als Admin of Leraar wil ik een leerling kunnen bekijken, zodat ik zijn gegevens tot mijn beschikking heb.
  • 3.1: Als Admin of Leraar wil ik nieuwe klassen aan kunnen maken, zodat ik kan zien welke leraren en leerlingen er in een klas zitten.
  • 3.2: Als Admin of Leraar wil ik klassen kunnen bewerken, zodat ik de klassen up-to-date kan houden.
  • 3.3: Als Admin of Leraar wil ik klassen kunnen verwijderen, zodat ik oude klassen weg kan gooien.
  • 3.4: Als Admin of Leraar wil ik een klas kunnen bekijken, zodat ik de details van deze klas kan bekijken.
